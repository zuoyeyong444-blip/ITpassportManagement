ITパスポート確認テスト（9-05 ～ 9-08）

９－０５ 労働関連・取引関連法規

1. 賃金や労働時間、休息、休暇など、労働者の労働条件の最低基準を定めた法律はどれか。

正解: D
解説：労働基準法は、労働条件の最低基準を定めた法律です。

2. 労働者と使用者が対等の立場で労働条件について合意し、労働契約を締結することを定めた法律はどれか。

正解: D
解説：労働契約法は、労働契約の締結や変更についてのルールを定めた法律です。

3. 派遣労働者を保護するための規定を定めた法律はどれか。

正解: C
解説：派遣労働者の権利と労働条件を保護するのは労働者派遣法です。

4. 公益のために内部告発した労働者が、解雇などの不利益な扱いを受けないように保護する法律はどれか。

正解: C
解説：不正を告発した人を守るのが公益通報者保護法です。

5. 所定の労働時間を満たしていれば、始業や就業時間を定時からずらしてよいとする制度はどれか。

正解: B
解説：フレックスタイム制は、始業・終業時刻を労働者が選べる制度です。

6. ICTを活用して、時間や場所の制約を受けない柔軟な働き方はどれか。

正解: D
解説：テレワークは、遠隔地からITを使って働くスタイルです。

7. 研究者やデザイナー、SEなどの専門的な業種対象で、労働者の裁量で労働時間を決められる制度はどれか。

正解: C
解説：特定の専門職において、実際の労働時間に関わらず、みなし労働時間を適用する制度です。

8. 事業運営、企画業務などに従事する人が対象となる、労働者の裁量で労働時間を決められる制度はどれか。

正解: C
解説：企業の根幹に関わる企画・立案を行う業務が対象です。

9. 労働者が、派遣元企業との雇用関係とは別に、派遣先企業の指揮命令を受ける契約はどれか。

正解: D
解説：派遣は、雇用主（派遣元）と指揮命令者（派遣先）が分離しています。

10. 将来直接雇用があることを前提とする形態の派遣はどれか。

正解: A
解説：一定期間（最長6ヶ月）の派遣後、双方が合意すれば正社員等として雇用されます。

11. 請け負った仕事を期日までに完成させることを約束して、成果物に対して対価を支払う契約はどれか。

正解: A
解説：請負は、成果物の「完成」が目的であり、指揮命令は受注者が行います。

12. 請け負った仕事が契約の内容に合わないときに負う責任はどれか。

正解: B
解説：納品物が契約内容と異なる（種類・品質・数量など）場合に負う責任です。

13. 製造物の欠陥が原因で人的被害が生じた場合の損害賠償の責任について定めた法律はどれか。

正解: B
解説：製品に欠陥があった場合、メーカーの過失を問わず被害者を救済する法律です。

14. 下請法改正により、適用対象が資本金基準から従業員数基準へ拡大し、運送取引も追加された法律はどれか。

正解: A
解説：親事業者の優越的地位の濫用を防ぎ、下請け業者を守る法律です。2026年1月1日施行。

15. 訪問販売や通信販売など、消費者トラブルが発生しやすい取引において消費者保護を目的とした法律はどれか。

正解: C
解説：クーリングオフ制度などが定められています。

16. 消費者を誤認させるような過大広告や虚偽表示を規制する法律はどれか。

正解: C
解説：実際よりも著しく優良、または有利だと偽る表示を禁じています。

17. 資源の分別回収や再資源化を推進するために制定された法律はどれか。

正解: D
解説：資源有効利用促進法などの総称です。

18. 電子マネーやプリペイドカードの発行・管理に関する法律はどれか。

正解: B
解説：IT技術を利用した決済サービスの安全性を守る法律です。

19. 有価証券や金融商品の発行・売買において、投資者保護や公正な取引を目的として定められた法律はどれか。

正解: C
解説：インサイダー取引の禁止などもこの法律に含まれます。

20. 国や自治体の行政機関が保有する資料の原則公開を義務付けた法律はどれか。

正解: D
解説：行政の透明性を確保するための法律です。

21. 不正な販売行為を防ぐために，正当な理由なく映像ソフトのコピープロテクトを無効化するプログラムの販売行為を規制している法律はどれか。

正解: D
解説：アクセスコントロールを回避する装置の提供は、不正競争防止法で禁じられています。

22. 労働者派遣における派遣労働者の雇用関係に関する記述のうち，適切なものはどれか。

正解: B
解説：雇用契約は派遣元会社と結びます。指揮命令を受けるのが派遣先です。

23. A社がB社に作業の一部を請負契約で委託している。発注者（A社）が直接労働者に指示を出すなど、実態が派遣と同じになっている違法な状態を何と呼ぶか。

正解: A
解説：請負なのに発注者が直接指示を出すことは禁じられています。

９－０６ 業務分析手法

24. 特性がどのような要因で引き起こされたかを、魚の骨のような図で体系的にまとめた図はどれか。

正解: D
解説：フィッシュボーンチャートとも呼ばれ、原因分析に使われます。

25. 項目を大きさの順に並べた棒グラフと、その累積比率を表す折れ線グラフを組み合わせた図はどれか。

正解: B
解説：重要な少数項目を特定するために使われます。

26. パレート図を利用し、項目をA・B・Cの3つのグループに分類して管理の重点を絞る分析手法はどれか。

正解: D
解説：在庫管理や売上分析などで、重要度に応じた管理を行う手法です。

27. 連続するデータをいくつかの区間に分け、各区間に属するデータの個数を棒グラフで表した度数分布図はどれか。

正解: C
解説：データのばらつき（分布）を視覚化します。

28. 最大値、最小値、中央値、四分位数など、データの分布を長方形の箱とひげで表した図はどれか。

正解: C
解説：複数のデータの分布を比較するのに適しています。

29. 中心線と管理限界線を引き、工程が安定しているかどうかを判定するための折れ線グラフはどれか。

正解: D
解説：製造現場などで異常検知に使われます。

30. 数値の大小を色の濃淡などで表現し、一目でデータの特徴を把握できるようにした表はどれか。

正解: B
解説：Webページのクリックエリア分析などでよく使われます。

31. 2種類のデータの相関関係を、縦軸と横軸の座標上にプロットして表した図はどれか。

正解: A
解説：右肩上がりなら正の相関、右肩下がりなら負の相関があると言えます。

32. 2種類のデータ間の関連性の強さを -1 から 1 の範囲で示す数値はどれか。

正解: D
解説：1に近いほど強い正の相関、-1に近いほど強い負の相関を意味します。

33. 回帰分析において、予測したい方の要素（結果となる値）を何と呼ぶか。

正解: A
解説：目的変数を、説明変数を使って算出します。

34. 回帰分析において、予測の根拠となる方の要素（原因となる値）を何と呼ぶか。

正解: D
解説：数式において目的変数を説明するための変数です。

35. 数式モデルを当てはめて、目的変数と説明変数の関係を明らかにする手法はどれか。

正解: C
解説：過去のデータから将来の値を予測する際などに用いられます。

36. 目的を達成するための手段を順次枝分かれさせて図解し、全体像を見える化する図はどれか。

正解: A
解説：問題を深掘りし、具体的な解決策を導き出すのに使います。

37. 二つの要素を行と列に分け、その交点に関連性を表示して整理した図はどれか。

正解: C
解説：多次元的な要素の組み合わせを整理します。

38. 多変数間の割合（構成比）を、長方形の面積で表現した図はどれか。

正解: A
解説：全体の構成と各項目の割合を同時に把握できます。

39. 概念地図と訳され、キーワード同士の関連を線などで表現して可視化する手法はどれか。

正解: A
解説：思考の整理や知識の構造化に役立ちます。

40. 部門横断的に全社員が参加して、製品やサービスの品質向上を目指す活動はどれか。

正解: A
解説：Total Quality Controlの略。日本で発展した品質管理手法です。

41. TQCを発展させ、経営戦略として全社的に継続的な品質向上を目指す手法はどれか。

正解: D
解説：Total Quality Managementの略。総合的品質経営です。

９－０７ データ利活用と問題解決

42. 統計調査において、分析の対象となる集団全体を何と呼ぶか。

正解: B
解説：日本国民全員の調査なら、日本国民が母集団です。

43. 母集団から一部のデータを抜き出して調査することを何というか。

正解: A
解説：サンプリングとも言います。

44. 母集団のすべての要素から、ランダムに標本を選ぶ抽出方法はどれか。

正解: B
解説：偏りがないようにランダムに選びます。

45. 母集団をいくつのかたまり（集団）に分け、その中から選んだ集団を全数調査する手法はどれか。

正解: D
解説：抽出の単位を集団にする方法です。

46. 片方の数値が増えるともう片方も増える、といった二者の関連性を何というか。

正解: A
解説：あくまで関連性であり、一方が原因とは限りません。

47. 二つのデータの間に「原因と結果」の関係があることを何というか。

正解: B
解説：一方が原因で、他方が結果として変動する関係です。

48. 非常に多くの項目があるデータを、情報の損失を抑えつつ少数の合成指標にまとめる手法はどれか。

正解: A
解説：次元を削減してデータの特徴を捉えやすくします。

49. 統計学で、あらかじめ立てた「差がない・関係がない」とする仮定を何と呼ぶか。

正解: A
解説：この仮説を否定（棄却）することで「意味がある」と結論づけます。

50. サンプルデータに基づき、ある仮説が正しいかどうかを確率的に判定する手法はどれか。

正解: B
解説：偶然起きたのか、必然なのかを判定します。

51. たくさんの具体的な事例から、共通する一般的な法則を導き出す推論はどれか。

正解: B
解説：AさんもBさんもCさんも死ぬから、人は死ぬ、と結論づける方法です。

52. 一般的な法則を前提として、個別の事例の結果を論理的に導く推論はどれか。

正解: C
解説：三段論法などが代表的です。

53. 特定の時点において、複数の調査対象（人や国など）を横断的に集めたデータはどれか。

正解: B
解説：ある一瞬を切り取って複数の対象を比べるデータです。

54. 批判厳禁、自由奔放など4つのルールを守り、多人数でアイデアを出し合う手法はどれか。

正解: B
解説：発散的思考を行い、アイデアの質より量を重視します。

55. 発言するのが苦手な人がいる場合などに、紙に書いて回覧しアイデアを広げる手法はどれか。

正解: A
解説：沈黙の中でアイデアを書き出し、他人のアイデアに乗っかります。

56. 会議において、中立的な立場で議論を促し、意見を整理して合意形成を助ける役割はどれか。

正解: C
解説：議事進行そのものをマネジメントします。

57. 昨日の気温が10℃、今日の気温が20℃のとき、「2倍暑い」という表現は数学的に正しいか。

正解: B
解説：摂氏温度（℃）は間隔尺度であり、0℃は「熱がない」ことではないため、比率計算はできません。

58. 0点が「全く存在しない」ことを意味し、身長や売上高などのように比率に意味がある尺度はどれか。

正解: B
解説：加減乗除がすべて可能な尺度です。

９－０８ 標準化

59. 工業や科学技術分野において、世界最大の国際規格を策定する機関はどこか。

正解: B
解説：国際標準化機構のことです。

60. 公的な機関が決めたものではないが、市場シェアなどで事実上の標準となったものはどれか。

正解: A
解説：キーボードのQWERTY配列などが有名です。

61. ISOなどの公的機関が、正当な手続き（法）に基づいて定めた標準はどれか。

正解: C
解説：法律や国際合意に基づいた標準です。

62. 特定の技術に関心を持つ複数の企業や団体が集まって策定する標準規格はどれか。

正解: C
解説：USBやWi-Fiの規格などはここに含まれることが多いです。

63. 日本国内において、産業製品やデータ、サービスなどの共通化を図るための国家規格はどれか。

正解: C
解説：日本産業規格（Japanese Industrial Standards）です。

64. 顧客満足度を高めるための「品質マネジメントシステム」に関する国際規格はどれか。

正解: A
解説：良い製品・サービスを安定して提供するための仕組みの規格です。

65. 企業活動による環境負荷を減らすための「環境マネジメントシステム」に関する国際規格はどれか。

正解: C
解説：環境保護や汚染予防を目指す仕組みです。

66. ITサービスを適切に管理・運用するための「ITサービスマネジメント」に関する国際規格はどれか。

正解: C
解説：ITIL（アイティル）をベースにした国際規格です。

67. 組織が守るべき「社会的責任（SR）」についてのガイドラインを示した国際規格はどれか。

正解: C
解説：認証用ではなく、ガイダンス（手引）として策定されています。

68. 情報の機密性・完全性・可用性を維持する「ISMS」に関する国際規格はどれか。

正解: D
解説：情報セキュリティマネジメントシステムの規格です。

69. 経営者がITの活用を適切に統制するための「ITガバナンス」に関する国際規格はどれか。

正解: A
解説：経営層によるIT活用のための原則を定めています。

70. 電気通信、無線通信など、通信分野の国際標準を策定する国連の機関はどれか。

正解: D
解説：国際電気通信連合（International Telecommunication Union）です。

71. クラウドサービスに関する情報セキュリティ管理を評価するための国際規格はどれか。

正解: C
解説：ISMSをクラウドサービスに適用するための指針です。

72. 日本独自の、個人情報保護マネジメントシステム（Pマークの基準）に関する規格はどれか。

正解: B
解説：プライバシーマーク制度の適合性審査の基準となる規格です。
